🐛 QUICK FIXES
================================================================================

Campaign not appearing in notifications?
  → Check user role matches campaign target
  → Check campaign is_active = true
  → Check RLS policies

Commission not calculating?
  → Check pieces_sold >= commission_min_qty
  → Verify offer_type = 'commission'
  → Check applySaleCommission is called

Wallet not updating?
  → Check wallet_transactions table in Supabase
  → Verify balance_after value exists
  → Restart dev server

Styles looking wrong?
